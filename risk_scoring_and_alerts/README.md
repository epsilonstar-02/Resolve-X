# Risk Scoring & Early Warning Service (RX-022 / RX-023)

This service computes risk scores for spatial clusters generated by the DBSCAN pipeline (RX-021) and provides early-warning alerts for department heads.

## Setup

1. Install dependencies:
   ```bash
   uv sync
   ```

2. Run the service:
   ```bash
   uvicorn main:app --reload --port 8020
   ```

## API Endpoints

- `GET /risk/zones`: Returns heatmap data with risk scores (0-100).
- `GET /risk/alerts`: Returns human-readable alerts for high-risk zones.
- `GET /healthz`: Health check endpoint.
